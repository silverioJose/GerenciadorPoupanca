# Gerenciador de Poupança

Aplicação desktop desenvolvida em JAVA com JavaFX para gerenciamento de cofres de poupança.

Permite criar, visualizar e excluir cofres, além de realizar depósitos e saques com controle de percentual atingido.

Desenvolvido como projeto pessoal para aprendizado de **JAVA** e **JavaFX**. 

---

## Roadmap

### Concluído
- Criar e excluir cofres
- Depósito e saque
- Visualização de saldo com toggle de privacidade
- Controle de meta e percentual atingido
- Persistência em banco de dados

### Em andamento
- Ajustes de interface
- Tela de últimas transações

### Planejado
- Tela de configurações
- Histórico completo
